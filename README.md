# Mini Backend Flask + MySQL 

Este proyecto es un mini-sistema backend desarrollado con Flask y MySQL. 
Simula la gestion basica de clientes y movimientos (CRUD), con el objetivo
de practicar conceptos fundamentales de desarrollo backend.

## Tecnologias utilizadas
Python
Flask
MySQL
HTML (formulario simple)
Git / GitHub

## Funcionalidades
Crear clientes
Listar clientes
Actualizar clientes
Eliminar clientes
Crear movimientos
Listar movimientos
Liistar movimientos por clientes 

## Objetivo del proyecto
Este proyecto fue creado con fines de aprendizaje para entender:
Como funciona un servidor backend
Que es una API REST
Como se comunican Flask y MySQL
Uso basico de control de versiones con Git 

## Como ejecutar el proyecto
Crear una base de datos MySQL
Configurar las credenciales en el archivo de conexion
